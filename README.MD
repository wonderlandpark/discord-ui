# Discord UI

Discord Styled Components

## Installation

### Clone

```sh
$ git clone https://github.com/wonderlandpark/discord-ui.git discord-ui
```

### Import

```html
<link rel="stylesheet" href="discord-ui/dui/dui.css">
```
